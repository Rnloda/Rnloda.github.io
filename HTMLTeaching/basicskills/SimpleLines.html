HTML document can be divided into paragraphs.

HTML paragraphs:

Paragraphs are defined by the<p> tag.

 

For example

<p>This is a paragraph</p>

<p>This is another paragraph</p>

Notes:

The browser automatically adds blank lines before and after the paragraph.（<p>Is a block level element）

Hins:

Using the empty paragraph tag<p> </p> to insert a blank line is a bad habit. Replace it with the </br>tag!

DO NOT FORGET ENDING TAG!!
