What is HTML?

HTML is a language used to describe web pages

  • HTML stand for Hyper Text Markup Language
  • HTML is not a programming language,but a markup language
  • Markup language is a sets of markup tag
  • HTML usemarkup tag to describe web pages

HTML tag

HTML lable tag is usually called as HTML tag

  • HTML tag is formed by a pair of <> which contains certain tags,such as <html>.etc
  • HTML usually works in pairs,like <b> and </b>
  • The first pair of two tags is Start tag,and the other one is End tag
  • Start and end tag is also named Open tag and Close tag

HTML Document = Webpage

  • HTML Documentdescribe webpages
  • HTML Documentinclude HTML tags and Plain Text
  • HTML Document is also called Webpage
